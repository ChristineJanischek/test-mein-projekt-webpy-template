# Legacy Analysis Bereich

Dieses Verzeichnis dient zur nachvollziehbaren Analyse von hochgeladenem Legacy-Code.

## Struktur
- source/
  - Entpackter Stand aus upload/webprojekt.zip

## Ablauf
1. Legacy-Quellstand in source ablegen oder entpacken.
2. Analyseergebnisse in docs/handbuch/legacy-analysis dokumentieren.
3. Relevante Fachlogik in Zielarchitektur überführen.
4. Änderungen im Pflichtenheft und Handbuch protokollieren.
